# Тема магистерской диссертации

## Формулировка темы

**«Семантическое структурирование и обогащение речевых описаний для генерации изображений средствами нейронных сетей»**

---

## Краткое описание

Работа посвящена разработке системы преобразования естественной человеческой речи в структурированные и обогащённые текстовые промпты, пригодные для использования с современными генеративными моделями изображений (Stable Diffusion, SDXL, Flux и др.).

### Ключевые аспекты:

1. **Семантическое структурирование** — анализ речевых описаний с выделением:
   - действий и событий;
   - объектов и персонажей;
   - пространственных отношений (локации, композиция);
   - эмоциональной окраски и атмосферы.

2. **Обогащение промптов** — трансформация базового текста в детализированные промпты:
   - добавление технических тегов для генеративных моделей;
   - включение стилистических дескрипторов;
   - учёт контекста сцены и истории взаимодействия;
   - персонализация на основе пресетов пользователей.

3. **Интеграция с генеративными моделями** — адаптация промптов под требования конкретных моделей (SD, SDXL, Turbo/LCM варианты).

---

## Цель работы

Разработать алгоритм и систему преобразования речевых описаний в обогащённые текстовые промпты, обеспечивающие высокое качество генерации изображений современными диффузионными моделями.

---

## Задачи исследования

1. Провести анализ существующих подходов к:
   - распознаванию речи (ASR);
   - семантическому анализу текста;
   - prompt engineering для генеративных моделей;
   - обогащению и структурированию промптов.

2. Разработать методы:
   - нормализации речевых описаний (очистка, исправление ошибок ASR);
   - семантического структурирования (выделение сущностей, действий, отношений);
   - обогащения промптов (добавление деталей, стилистических маркеров, технических тегов).

3. Реализовать прототип системы, включающий:
   - модуль потокового распознавания речи;
   - модуль семантического анализа и структурирования;
   - модуль обогащения промптов (rule-based и/или LLM-based подходы);
   - интеграцию с генеративными моделями (Stable Diffusion и варианты).

4. Провести экспериментальную оценку:
   - качества преобразования речи в промпты (метрики точности, полноты);
   - влияния обогащения промптов на качество генерируемых изображений;
   - сравнение различных подходов к структурированию и обогащению.

---

## Научная новизна

- Разработка методов адаптации естественной речи к формату промптов генеративных моделей с учётом семантической структуры описаний.
- Исследование влияния различных стратегий обогащения промптов на качество генерации изображений.
- Создание архитектуры потоковой обработки речевых описаний для интерактивных сценариев использования.

## Технологический стек

- **Язык программирования**: Python 3.x
- **ML-фреймворки**: PyTorch, Hugging Face Transformers, Diffusers
- **ASR**: faster-whisper, Vosk (для сравнения)
- **Семантика/обогащение**: 
  - Rule-based подходы (шаблоны, регулярные выражения)
  - LLM-based (опционально: небольшие модели или API)
- **Генерация изображений**: Stable Diffusion (SD 1.5, SDXL, Turbo/LCM варианты)
- **Архитектура**: FastAPI (микросервисы), MLflow (трекинг экспериментов)
- **Инфраструктура**: Jupyter Notebooks (прототипирование), Docker (опционально)
